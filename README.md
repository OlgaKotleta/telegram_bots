# Telegram Echo Bot

Простой эхо-бот на чистом Python без фреймворков.

### Текущая реализация (Д/З #5)
-  **Эхо-бот** - отвечает на текстовые сообщения и фото
-  **Логирование** - сохраняет все апдейты в SQLite базу
-  **Архитектура** - паттерн Dispatcher с цепочкой обработчиков
-  **Обработка фото** - отправляет эхо-фото (самую большую версию)

### Реализованные обработчики:
1. **DatabaseLogger** - логирует все апдейты в БД
2. **MessageTextEcho** - эхо-ответ на текстовые сообщения  
3. **MessagePhotoEcho** - эхо-ответ на фото

## Установка
1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать: `source venv/bin/activate`
4. Установить зависимости: `pip install -r requirements.txt`
5. Создать файл `.env` с токеном бота: `BOT_TOKEN=your_token_here`
6. Запустить: `python bot.py`
