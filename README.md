# Telegram Echo Bot

Простой эхо-бот на чистом Python без фреймворков.

## Функциональность
- Отвечает эхом на любое текстовое сообщение
- Сохраняет все входящие сообщения в SQLite базу данных
- Работает через long polling

## Установка
1. Клонировать репозиторий
2. Создать виртуальное окружение: `python -m venv venv`
3. Активировать: `source venv/bin/activate`
4. Установить зависимости: `pip install -r requirements.txt`
5. Создать файл `.env` с токеном бота: `BOT_TOKEN=your_token_here`
6. Запустить: `python bot.py`
